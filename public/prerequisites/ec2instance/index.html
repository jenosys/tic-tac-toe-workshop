워크샵에 IAM 역활 연결하기

  1. 다음 링크를 누르세요 Cloud9 EC2 instance 화면
  2. 인스턴스를 선택하고, 다음을 선택합니다. Actions / Instance Settings / Attach/Replace IAM Role
  3. IAM Role 항목을 tic-tac-toe-workshop-admin 로 선택하고, Apply 를 선택합니다.
